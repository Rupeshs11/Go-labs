## 🧠 About Go

**Go** (Golang) is an open-source programming language created at **Google** in 2009 by Robert Griesemer, Rob Pike, and Ken Thompson.

### Why Go?

- ⚡ **Fast compilation** — compiles to native machine code
- 🧵 **Built-in concurrency** — goroutines & channels make parallel programming easy
- 📦 **Simple dependency management** — `go mod` handles everything
- 🔒 **Type safe** — statically typed with garbage collection
- 🚀 **Single binary deployment** — `go build` produces one executable, no runtime needed
- 🌐 **Great standard library** — HTTP servers, JSON, crypto, file I/O — all built-in

### Who Uses Go?

Docker, Kubernetes, Terraform, Prometheus, Grafana, Hugo, CockroachDB, and many more.
